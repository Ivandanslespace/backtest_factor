# Instructions du projet

- Répondre toujours à l'utilisateur en chinois.
- Rédiger en français tous les commentaires, docstrings et messages ajoutés aux fichiers Python.
- Après chaque modification validée, créer automatiquement un commit puis le pousser vers `origin/main`.
- Ne jamais versionner les fichiers `*.parquet`, le dossier `exports/`, les caches Python ni les sorties temporaires des notebooks.
- Avant chaque push, vérifier explicitement la liste des fichiers suivis et supprimer les sorties d'exécution intégrées aux notebooks.
